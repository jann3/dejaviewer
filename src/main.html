Help

Keyboard Shortcuts

Press Escape to return to the start page at any point, pressing it again will then close the app.

For keyboard navigation, use Tab to jump between actions and Space / Enter to select.

You can use CTRL + O to open the file browser, and F11 to enter fullscreen mode.
